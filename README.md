Otus. Индустриальный проект.
===============================================

Описание
------------
Веб-сервис для планирования количества ресурсов и аппаратных хостов на базе поступающих заявок.
Приложение ведет список аппаратных хостов, принимает заявки на выдачу виртуальных машин и позволяет назначать заявки на ресурсы.

Функционал администратора:
 - Добавлять новые хосты
 - Редактировать имеющие хосты: 
    + Изменение статуса хоста
    + Изменение объема ОЗУ
    + Добавление или удаление хранилища
    + Изменение ЦОД / сегмент сети / систему виртуализации
 - Получить все хосты и их нагруженность
 - Получить хосты в состоянии "В рассмотрении"
 - Отклонить заявку с указанием причины
 - Назначить заявку на конкретный хост

Функционал пользователей:
 - Создание заявки на виртуальную машину с указанием необходимых ресурсов
 - Отправить свою заявку на рассмотрение (изменить статус на "в рассмотрении") или отозвать (статус "создано")
 - Получить свои заявки и их статусы

Подробное описание доступных методов, параметров и схем, а так же отправка запросов доступно по адресу запущенного сервера (Swagger UI)
```
http://[server_ip:server_port]/docs
```
Установка зависимостей
------------
```
pip install -r requirements.txt
```
Запуск сервера
------------
```
python app.py
```


Совместимость
-------------
Python 3.6 +
